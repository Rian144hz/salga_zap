# Salga-Zap - Sistema de Pedidos via WhatsApp

## Descrição do Projeto
O Salga-Zap é um cardápio digital web desenvolvido para automatizar e organizar o recebimento de pedidos de uma salgadeira autônoma. Através da interface, o cliente escolhe os produtos, define as quantidades e visualiza o valor total da compra em tempo real. Ao finalizar, o sistema estrutura os dados do pedido e redireciona o usuário para o WhatsApp do estabelecimento com a mensagem pronta para o envio.

## Motivação
Apoiar a digitalização de micronegócios familiares que não possuem estrutura ou orçamento para manter plataformas de delivery pagas. O projeto aplica conceitos acadêmicos de manipulação do DOM e lógica de programação em JavaScript para resolver um problema real de atendimento e logística de pedidos de forma simples e direta.

## Funcionalidades
* Cardápio interativo com seletores dinâmicos de quantidade.
* Listagem de itens selecionados e cálculo automatizado do valor total.
* Integração com a API de links do WhatsApp para envio do pedido formatado.

## Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript Vanilla (ES6+)
